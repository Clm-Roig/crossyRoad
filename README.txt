---------------------------------------------------------------

Pour jouer, cliquer sur Run. La partie commence lorsque vous vous déplacez une première fois avec les flèches du clavier. 
Avancer vous ocroît 1 point, les boules dorées 5 points. 

Vous mourrez quand : 
- vous percutez une voiture
- vous percutez un train (les trains sont annoncés par des feux tricolores sur les côtés de la carte)
- vous tombez dans l'eau
- le défilement automatique rattrape votre personnage et vous sortez de la map

Quand vous mourrez, votre score s'affiche et vous pouvez recommencer une partie en appuyant sur "Entrée". 

REMARQUE : Il est possible d'ajuster la possibilité du jeu en diniminuant la "Speed" dans l'interface Greenfoot.
